Typography

Font Families

To use custom Google Web Fonts, go to google.com/webfonts and pick your font. In the instructions section it will show you the address of the font, e.g 'http://fonts.googleapis.com/css?family=Averia+Libre:700' - enter the font part (eg: 'Averia+Libre:700') into the input, select 'Custom' and the template will do the rest.

Homepage setup

Row ordering

The homepage is designed in rows, and you are able to place these in any order you desire, allowing for a super flexible layout. Omitting any letter will automatically prevent that row from displaying.

  • A: Text Row
  • B: Slideshow
  • C: Products
  • D: Adverts
  • E: Second Text Row
  • F: Blog
  • G: Custom HTML

Enter the letter of each row in the order you want them to show, separated with a hyphen.

E.g. D-C-A-B-E
